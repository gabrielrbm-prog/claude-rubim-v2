Gere um **componente premium** para: $ARGUMENTS

## Processo:

### 1. Busque inspiração
Use o Magic MCP (21st.dev) com `21st_magic_component_inspiration` para encontrar componentes similares de alta qualidade.

### 2. Gere o componente
Use `21st_magic_component_builder` para criar o componente com:
- React + TypeScript
- Tailwind CSS
- Framer Motion para animações sutis
- Responsivo (mobile-first)
- Acessível (ARIA, keyboard nav, focus management)

### 3. Garanta qualidade premium
Aplique estes critérios anti-"AI slop":
- **Cores**: paleta intencional, não gradientes genéricos azul-roxo
- **Tipografia**: hierarquia clara, font weights variados
- **Espaçamento**: sistema de 4/8px, consistente
- **Animações**: sutis (200-300ms), com reduced-motion support
- **Sombras**: naturais (não brilhantes/neon)
- **Border-radius**: consistente no componente
- **Dark mode**: suporte nativo com Tailwind dark:

### 4. Output:
- Componente TypeScript completo
- Props interface documentada
- Variantes (sizes, colors, states)
- Exemplo de uso
- Preview no Playwright (screenshot)
